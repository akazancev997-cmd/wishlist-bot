# 🎁 WishlistBot

**Telegram-бот для создания и управления вишлистами (списками желаний).**

Создавай списки желаний на день рождения, свадьбу, Новый год — делись ссылкой с друзьями и получай только те подарки, которые действительно хочешь!

---

## 🚀 Быстрый старт

### 1. Создай бота в Telegram

1. Открой [@BotFather](https://t.me/BotFather)
2. Напиши `/newbot` и следуй инструкциям
3. Сохрани полученный **токен** (выглядит как `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`)

### 2. Настрой переменные окружения

```bash
cp .env.example .env
# Отредактируй .env:
# BOT_TOKEN = токен от BotFather
# WEBHOOK_URL = https://твой-домен.ру (для продакшена)
```

### 3. Установи и запусти

```bash
# Установка зависимостей
pip3 install -r requirements.txt

# Запуск (локальная разработка)
python3 main.py

# Запуск (продакшен)
uvicorn web:app --host=0.0.0.0 --port=8080
```

## 📦 Развертывание (деплой)

### На [Render](https://render.com) — бесплатно

1. Зарегистрируйся на Render
2. Нажми **New + → Web Service**
3. Подключи GitHub-репозиторий
4. Настрой:
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `uvicorn web:app --host=0.0.0.0 --port=$PORT`
5. Добавь переменные окружения:
   - `BOT_TOKEN` = твой токен
   - `WEBHOOK_URL` = `https://твой-сервис.onrender.com`
6. Нажми **Create Web Service**

После деплоя:
```bash
# Установи вебхук (сделает бот автоматически):
# Просто открой https://твой-сервис.onrender.com
```

### На Railway / Heroku (аналогично)

Просто подключи репозиторий и пропиши те же переменные.

## 💰 Монетизация

### Премиум-подписка

| Тариф | Цена | Что даёт |
|-------|------|----------|
| Бесплатный | 0 ₽ | 1 вишлист, 10 желаний |
| Премиум | 249 ₽/мес | Безлимит всего, импорт с маркетплейсов, аналитика |
| Премиум (год) | 1 499 ₽/год | То же + скидка 50% |

### Платежи через ЮKassa

1. Зарегистрируй [ЮKassa](https://yookassa.ru) (бывш. Яндекс.Касса)
2. Получи `shopId` и `secretKey`
3. Добавь в `.env`:
   ```
   YOOKASSA_SHOP_ID=твой_shop_id
   YOOKASSA_SECRET_KEY=твой_секретный_ключ
   ```
4. Настрой вебхук в личном кабинете ЮKassa:
   - URL: `https://твой-домен/yookassa/webhook`
   - События: `payment.succeeded`, `payment.waiting_for_capture`
